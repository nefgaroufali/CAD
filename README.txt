TCL_Shell

This is a simple TCL shell implemented in C.

Prerequisites:
Ensure you have `gcc` installed on your system. You also need `tcl8.6` and `readline` libraries.


Compilation:
The project uses a Makefile for compilation. To compile the project, navigate to the project directory and run the following command in your terminal:

$make

This will create an executable named `TCL_Shell`.


Running the Program:
After successful compilation, you can run the program using the following command:

$./TCL_Shell


Cleaning Up:
To clean up the compiled files (object files and executable), use the following command:

$make clean
